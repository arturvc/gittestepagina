Exercício 3

Usando o GitHub para hospedar página web

Exercício de criação e hospedagem de página na world wide web no GitHub. Segundo o The Coding Train, primeiro é necessário criar um branch gh-pages, em seguida seleciona como branch default. Pra evitar confusão de enviar arquivos para o branch errado, convém deletar o master branch. Em Settings é possível ver a URL da página web, que segue essa estrutura: https://github.com/nomeDoUsuário/nomeDaPágina.

Após a realização desses procedimentos, o próximo passo é criar os arquivos da página web, ou subir (push) os já existentes. A página web do GitHub é um repositório como qualquer outro, e os comando de commit, pull, push, e etc continuam disponíveis, tanto pela interface web como pelo Git Bash.
